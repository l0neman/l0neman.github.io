Git 常用命令

初始化

  • 首次安装后设置名字和 email 地址。
  • 创建 ssh Key。
  • 进入目录并初始化为 git 仓库。

文件更改

  • 放弃文件在工作区的修改。
  • 添加文件到暂存区。
  • 撤销存放到暂存区的文件。
  • 查看文件修改部分。
  • 提交文件至仓库。
  • 删除并添加到缓存区。

仓库状态

  • 查看当前仓库状态。
  • 查看提交日志。
  • 查看命令执行历史记录。

版本控制

  • 回退版本, ^ 表示回退到上一个版本， ^^ 表示上上一个，以此类推。

分支管理

  • 创建并切换到目标分支。
  • 切换到目标分支。
  • 查看分支。
  • 查看分支合并图
  • 整理分支提交历史为直线。
  • 合并目标分支到当前分支，相当于指针直接指向目标分支。
  • 合并目标分支到当前分支
    并禁用 Fast forward 模式（非指针指向，而是提交和目标分支同样的内容）。
  • 删除目标分支。

现场保存

  • 保存当前工作现场。
  • 查看保存的 stash。
  • 恢复同时删除 stash 内容。
  • 恢复 stash 内容。
  • 删除 stash 内容。

标签使用

  • 为当前提交创建标签。
  • 为目标提交记录（通过指定 commit id）创建标签。
  • 列出所有 tag。
  • 查看目标 tag 详情。
  • 删除标签。
  • 推送标签至远程库
  • 从远程库删除标签。

远程库

  • 添加本地库到远程库。
  • 首次推送到远程库。
  • 推送到远程库。
  • 克隆远程库到本地。
